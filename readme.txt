Display messages notifications on your site. You can put your message on top or bottom of your site, and you can setup multiple messages.  

== Description ==

If you need to display short messages or notifications on your site, this is the right plugin for you. You can position your messages on top or bottom of your site, and show multiple messages at once if you like.

Customizable Settings:

* Message positioning: Top, bottom
* Set number of seconds to wait before message appear
* Set number of seconds to wait before the message disappear
* Messages can appear on site with your specified date
* Input CSS to customize the look and feel further 

Note: This version doesn't work with theme that don't call wp_footer

== Installation ==

To install this plugin:

1. Upload the folder `message1977` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
